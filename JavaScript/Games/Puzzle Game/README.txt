# We Puzzle It!

This is a multi-player game that serves as a PoC demo for various advanced HTML5 js APIs. It's is majorly a work-in-progress and has lots of bugs and unfinished things.

## Where To Play?

When the game is live (not often, at the moment, during development), it can be found at:

http://test.getify.com/wepuzzleit/
