Welcome to TextBoxify's documentation!
======================================

TextBoxify is a package for games created with `Pygame`_ that makes it easy to create text and dialog boxes.
It aims to be highly customizable so the user can design dialog boxes to their own taste.

Installation
------------

The package is available on PyPI and can be installed from the command line with::

    pip install textboxify

.. _Pygame: https://www.pygame.org/

How to use
----------

To use the package in your own project simply import it in your modules like this::

    import textboxify
