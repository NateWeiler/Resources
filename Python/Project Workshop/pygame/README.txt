# Pre-reqs

## Python 2.6 or higher installed

NOTE: Those with Linux and MacOSX would have Python installed by default, no action required.

Windows: Download the version for your laptop via [https://www.python.org/downloads/](https://www.python.org/downloads/)

## PyGame 1.91 installed
Download from [http://www.pygame.org/download.shtml](http://www.pygame.org/download.shtml)
