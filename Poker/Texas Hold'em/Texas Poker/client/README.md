poke-game-front-ts
=====================
## Setup

```
yarn
yarn serve
```
### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

## Project structure
```
├─public  // html
└─src
    ├─assets
    │  ├─less
    │  └─mp3
    ├─components
    ├─interface
    ├─plugins
    ├─router
    ├─service     // api
    ├─store
    ├─utils
    └─views

```
